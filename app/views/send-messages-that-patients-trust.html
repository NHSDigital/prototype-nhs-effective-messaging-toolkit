Send messages that patients trust

To help patients act on the messages we send them, it’s important that they can trust that messages are safe, secure and have genuinely been sent by the NHS.

Sender names on text messages

The text address which appears on a text message in place of the sending telephone number is called a SenderID.

Using an appropriate SenderID can help build trust with message recipients.

You should use MEF-registered (official) SenderIDs.
