Communication channels for messages

Most messaging service suppliers will only let you decide which communication channel to use to message an individual patient. They will have fixed channels for sending batch messages to multiple patients.

Avoid sending the same message using multiple channels at the same time. This can overwhelm patients. It also increases costs for the NHS.

Send messages that patients can understand

If a patient with a disability, impairment or sensory loss has a recorded communication need, you must send messages in a format they can access.

Find out more about sending accessible and inclusive messages.

Understand the benefits of different channels

Different communication channels may be more appropriate for different types on messages. You should also consider how much sending a message will cost.

Find out more about:

  • NHS App messages
  • text messages (SMS)
  • emails
  • paper letters

Create a communications policy

Create a communications policy that outlines what types of messages will be sent using each channel.

This will make your messaging more consistent for patients, so they know what to expect. It will also help you budget for messaging costs.

NHS England has created a template for email and text message communications.
