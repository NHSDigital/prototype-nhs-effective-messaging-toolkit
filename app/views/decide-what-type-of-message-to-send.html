Decide what type of message to send

Most [message services] will only let you decide what time of message to send for individual messages. They will have a set priority order for [batch] messages.

Avoid sending the same message using multiple channels at the same time. This can overwhelm patients. It also increases costs for the NHS.

Send messages that patients can understand

If a patient has a recorded communication need, you must send messages in a format they can access. Find out more about sending accessible messages

App messages

Sending a message through the NHS App is free and secure. It also means patients can manage all their healthcare information in one place. Find out more about the benefits of NHS App messaging.

Patients need to download the NHS App and turn on their notifications before they can receive messages through the app. Find out more about the benefits of NHS App messaging.
